# Shadcn/ui Components to Install

Die folgenden Komponenten werden für die CRUD-Pages benötigt und sollten per CLI installiert werden:

- `table`
- `dialog`
- `tabs`
- `select`
- `textarea`
- `toast`
- `skeleton`
- `switch`
- `form`
- `popover`
- `command`
- `scroll-area`

## Installation

Verwenden Sie die Shadcn CLI, um jede Komponente hinzuzufügen:

```bash
npx shadcn@latest add <component>
```

Die CLI erzeugt die benötigten Dateien unter `src/components/ui/`.

> Hinweis: Diese Datei dient lediglich als Referenzliste. Die eigentliche Installation erfolgt manuell über die CLI.
